# dwm-night&day
Changes between night and day with the script included xd <br>
Peak programing over here

# Dependencies
- libxft-bgra



# Steps for installation (easy)

```git clone https://github.com/Mattio-cmd/dwm-night-day```

```cd dwm-night-day```

If its the first time you are installing dwm then do this:

```doas make```

If not, then:

```doas make clean install```

Thats it, just add it to your xprofile & xinitrc, or if you use a login manager add the dwm.desktop
